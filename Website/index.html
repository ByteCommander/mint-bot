Add Bot to your server

  • Clone or download the bot files
  • Download and Install node.js and NPM (usually comes in one package)
  • Install build-essentials trough your regular package manager
  • Download and install Python 2.7
  • Download and Install ffmpeg
  • Configure npm and install the project dependencies
      $ npm config set python python2.7
      $ npm install discord.js fs canvas request node-opus simple-youtube-api ytdl-core node-ddg translate-google moment better-sqlite3 rss-emitter-ts curl html-to-text steam-searcher nodejs-linenumber
  • Inside the file ./set/prefix.txt change the content to:
      !
  • In the file ./config.json change the tokens/keys
  • To start the bot simple start a terminal in your bot folder and type:
      while true; do node index.js; done
  • If your server is completely new, then you can allow Artemis to setup herself by typing in the discord channal chat:
      setup auto
    • These are some essential commands you type in your Discord channel where Artemis is.
  • !channelmanage
      This will allow you to change and check the setup channels. Artemis will use a various amount of channels to display different content.
  • !levelmanage
      This will allow you to set level up roles.
  • !rolemanage
      To make !join and !leave usable you have to add roles to the database, so that it can be used. You can also use this in combination with reaction roles: If the role name and emote name are exactly the same then if Artemis has a reaction channel setup in there you can add the emote to a message to make it a reaction role. reactions for custom emojis are made with !react
  • !streamping
      If you have a streams channel setup (for twitch) then you can use this command !streamping on/off to enable @here notifications.
    • Artemis has a few features that may be handy to you.
  • !automod
      When this is turned on it will enable a word filter (setup with !wordlist add word word), anti mention spam, anti message spam (1000ms) and anti discord invite links. Mods and Admins and everyone with the kick permission do not get affected by the automod.
  • Streaming channel
      When you setup a streaming channel (!channelmanage stream chanID) Artemis will catch every person who changed their status to their twitch stream. users can turn this off by using !stream off
  • Reaction roles
      If a reaction channel has been setup (!channelmanage reaction chanID) you can add emotes to a message in that channel that are IDENTICAL to the role names you want to be available. You setup available roles with !rolemanage and you add emotes to a message with !react word word Do note that !react only accepts custom emotes!
  • New member filter
      If you have a mute channel setup and a muted role(muted) and a member role(~/Members) then Artemis will catch all accounts that are below 30 days old and makes them go trough a verification process.
  • Warn system
      using !warn @user will warn the specified user, when this user gets 3 warns he will get muted (regardless if a mute channel has been set up). If a mute channel has been set up then the user will only have access to that single channel. using !warn @user REASON will add a reason to their warn. you can check this with !warnings or !warnings @user
  • Logs
      If you have set up a logs channel then this channel will display all mod/admin command usage as well as nickname/username changes and user leaves/joins.
