4 5 1. Install XFCE.

apt (Debian, Ubuntu, Linux Mint, Pop!_OS): sudo apt install xfce4

yum (RHEL 7, CentOS): 
sudo yum install epel-release -y && sudo yum groupinstall "Server with GUI" -y && sudo yum groupinstall "Xfce" -y

dnf (Fedora, RHEL 8): sudo dnf install @xfce-desktop-environment

2. Reboot your system.

3. At your login manager, change your desktop environment to XFCE.

If you need help with using XFCE, consider checking the XFCE documentation.
https://docs.xfce.org/start
