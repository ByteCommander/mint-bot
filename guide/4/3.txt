4 3 1. Install GNOME 3.

apt (Debian, Ubuntu, Linux Mint, Pop!_OS):
sudo apt-add-repository ppa:gnome3-team/gnome3
sudo apt-get update
sudo apt-get install gnome-shell

yum (RHEL, CentOS): sudo yum groupinstall "GNOME Desktop"

eopkg (Solus):
sudo eopkg rm lightdm
sudo eopkg install gdm gnome-shell gnome-desktop-branding-fortitude

2. Reboot your system.

3. At your login manager, change your desktop environment to GNOME 3.

If you need help with using GNOME applications, consider checking GNOME Help.
https://help.gnome.org/users/gnome-help/stable/
