4 4 1. Install i3.

apt (Debian, Ubuntu, Linux Mint, Pop!_OS): sudo apt-get install i3

yum (RHEL 7, CentOS): sudo yum --enablerepo=extras install epel-release && sudo yum install i3

eopkg (Solus): sudo eopkg install i3

dnf (Fedora, RHEL 8): sudo dnf install i3

2. Reboot your system.

3. At your login manager, change your desktop environment to i3.

If you need help with using i3, consider checking the i3 User's Guide.
https://i3wm.org/docs/userguide.html
