4 2 1. Install KDE Plasma.
apt (Debian, Ubuntu, Linux Mint, Pop!_OS): sudo apt install kde-plasma-desktop
yum (RHEL, CentOS): sudo yum groupinstall -y "KDE Plasma Workspaces"
dnf (Fedora): sudo dnf -y group install "KDE Plasma Workspaces"
eopkg (Solus): sudo eopkg install -c desktop.kde.core

2. Reboot your system.

3. At your login manager, change your desktop environment to KDE Plasma.

If you need help with using KDE applications, consider checking the KDE UserBase Wiki.
