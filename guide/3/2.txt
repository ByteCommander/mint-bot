3 2 Virtualbox is a program that allows you to create virtual machines.
A virtual machine (VM) is a virtual environment that functions as a virtual computer system with its own CPU, memory, network interface, and storage.

1. Install Virtualbox.

apt (Debian, Ubuntu, Linux Mint, Pop!_OS): sudo apt install virtualbox

yum (RHEL 7, CentOS): cd /etc/yum.repos.d && sudo wget http://download.virtualbox.org/virtualbox/rpm/rhel/virtualbox.repo && sudo yum --enablerepo=epel install dkms && sudo yum groupinstall "Development Tools" && sudo yum install kernel-devel && sudo yum install VirtualBox-5.2 && usermod -a -G vboxusers $USER

dnf (Fedora, RHEL 8): sudo dnf -y install @development-tools && sudo dnf -y install kernel-headers kernel-devel dkms elfutils-libelf-devel qt5-qtx11extras && wget https://pastebin.com/raw/8a7NcUXE && ./8a7NcUXE && sudo dnf search virtualbox && sudo dnf install VirtualBox-6.0

eopkg (Solus): sudo eopkg install virtualbox-current

2. You can create a VM, import a VM, export a VM, add an existing VM image, or change your preferences on the right side of the window, above the text "Welcome to Virtualbox!". When creating a VM, slightly increasing the default amounts of RAM and storage will provide a better experience.

Once you create a VM, you can run it by clicking on it under the "Tools" button, then clicking the "Start" button.

You can access the settings and change the configuration of a VM by clicking on the "Settings" button next to the "Start" button.

If you need help with using Virtualbox, consider checking the Virtualbox Manual.
https://www.virtualbox.org/manual/
